Lo que hago.

Con el fin de prestar servicio a mis clientes, he diseñado diferentes etapas que permiten una comunicación dinámica (cliente-promotor) con el objetivo de crear confianza y que el resultado cumpla con sus expectativas en este sentido:

01.
Diseño web

De acuerdo con las expectativas y la descripción proporcionada por el cliente, diseñaré el sitio web, y enviaré la propuesta correspondiente para su aprobación.

02. 
Funcionalidad

El cliente describe las funciones que deben ir en la página, como los vídeos, las imágenes y el texto, que pueden ser realizadas por el cliente o por un servicio externo.

03.
Desarrollo web

Las páginas se desarrollarán en HTML, CSS y JavaScript, y se entregarán a tiempo.
